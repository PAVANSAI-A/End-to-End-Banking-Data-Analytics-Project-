# 🏦 End-to-End Banking Data Analytics Project

> 📊 Built using Python, MySQL, and Power BI to analyze customer financial behavior and generate business insights

---

## 🎯 Problem Statement

Banks need to understand customer financial behavior to improve profitability, identify high-value customers, and support data-driven decision-making.

This project focuses on analyzing banking data to:
- Understand customer segmentation  
- Analyze loan and deposit trends  
- Identify relationships between financial variables  
- Generate actionable business insights  

---

## 🚀 Project Overview

- Built an end-to-end banking analytics solution using **Python, MySQL, and Power BI**  
- Analyzed a dataset of **3,000 customers with 24 financial features**  
- Designed a workflow from data extraction to dashboard visualization  

---

## 🛠 Tech Stack

- Python (Pandas, NumPy, Seaborn, Matplotlib)  
- MySQL  
- Power BI  
- Google Colab  

---

## 🔍 Methodology

### Data Preparation
- Loaded dataset into MySQL  
- Connected MySQL with Python using connector  

### Exploratory Data Analysis
- Used `info()`, `shape()`, `describe()` for data understanding  
- Performed univariate and bivariate analysis  

### Statistical Analysis
- Conducted correlation analysis using heatmap  
- Identified relationships between financial variables  

### Visualization
- Built interactive Power BI dashboard for business insights  

---

## 📊 Key KPIs

- Total Loans: **~₹4.38 Billion**  
- Total Deposits: **~₹3.77 Billion**  
- Business Lending: **~₹2.60 Billion**  
- Checking Accounts: **~₹963 Million**

## 📈 Key Insights

- Medium-income customers contribute **over 50% of total loans and deposits**  
- Strong positive correlation (**0.84**) between **Bank Deposits and Checking Accounts**  
- Customers with high balances tend to maintain **multiple account types**  
- Deposit and account balances indicate presence of **high-value customer segments**
